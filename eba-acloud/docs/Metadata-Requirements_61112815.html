Metadata Requirements

The following table contains a list of the metadata properties required for Skyscape (VmWare based) and AWS.  When updating a design and in the event you may have missed a property, when the export button is pressed, you will be notified as to which property is required. This will differ depending on whether you have selected Skyscape or AWS as the provider in the Environment Container.  There may be a design which may not have some properties that are mentioned in the below table; over time the stencil has been improving, therefore removing some properties which are not necessary in the design but can be populated within the export macro.  However, continue as normal and populate as necessary to avoid confusion.
