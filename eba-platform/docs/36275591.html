VPC's and VPC Peer Vlans

An AWS VPC maps to an Environment Group or Environment Container. This means all VMs in a VPC will have the same first 2 octets in the IP address. The interfaces between VPCs require Peer Vlans. Connectivity is achieved by connecting a vlan from each VPC together.

 

The vlan definitions in the physical designs will need extra attributes added to them.

Peer_Remote_VPC     (e.g. HO_IPT_NPA_TLG1)

Peer_Remote_vLAN    ( e.g. HO_IPT_NP_TLG1_MEBC_APP_I1)

 

Designs will need to change so that each end of a cross VPC vlan is defined as a separate lan using the local address range. These vlans are then joined together.using the VPC peering.
